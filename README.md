# ⚡ 전기기사 핵심 개념 학습 플랫폼

전기기사 자격증 시험을 준비하는 수험생을 위한 **완전 무료** 온라인 학습 플랫폼입니다.
110개의 핵심 개념을 6개 과목으로 체계적으로 정리하여 언제 어디서나 편하게 학습할 수 있습니다.

## 🎯 주요 기능

### 📚 체계적인 학습 구조
- **6개 핵심 과목**: 전기이론, 전력공학, 전기기기, 회로이론, 제어공학, 전기설비기술기준
- **110개 핵심 개념**: 각 개념마다 정의, 공식, 예시, 관련 개념 포함
- **출제빈도 표시**: 상/중/하 빈도로 중요도 파악

### 🔍 강력한 검색 및 필터
- **실시간 검색**: 개념, 공식, 용어를 즉시 검색
- **카테고리 필터**: 과목별로 필터링
- **빈도 필터**: 출제빈도별로 필터링
- **학습 상태 필터**: 전체/즐겨찾기/학습완료/미학습 구분

### ⭐ 개인화 학습 관리
- **즐겨찾기**: 중요한 개념을 즐겨찾기에 추가
- **학습 진도 추적**: 학습한 개념을 체크하고 진도율 확인
- **로컬 저장**: 브라우저에 자동 저장되어 다시 방문해도 유지

### 📱 완벽한 모바일 지원
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 최적화
- **다크모드**: 눈이 편한 다크모드 지원
- **PWA 지원**: 홈 화면에 추가하여 앱처럼 사용 가능

### 🎨 사용자 경험
- **모던한 UI**: 직관적이고 깔끔한 인터페이스
- **빠른 로딩**: 정적 웹사이트로 초고속 로딩
- **오프라인 지원**: 한 번 방문하면 오프라인에서도 사용 가능

## 🚀 GitHub Pages 배포 가이드

### 방법 1: GitHub 웹 인터페이스 사용 (가장 쉬움)

1. **GitHub 계정 만들기**
   - https://github.com 에서 무료 계정 생성

2. **새 저장소 만들기**
   - 우측 상단 `+` 버튼 → `New repository` 클릭
   - Repository name: `electrician-study` (원하는 이름)
   - Public 선택 (무료 호스팅을 위해 필수)
   - `Create repository` 클릭

3. **파일 업로드**
   - `uploading an existing file` 링크 클릭
   - 다음 파일들을 드래그하여 업로드:
     - `index.html`
     - `styles.css`
     - `script.js`
     - `data.js`
     - `README.md`
   - `Commit changes` 클릭

4. **GitHub Pages 활성화**
   - 저장소의 `Settings` 탭 클릭
   - 왼쪽 메뉴에서 `Pages` 클릭
   - Source에서 `main` 브랜치 선택
   - `Save` 클릭
   - 몇 분 후 `Your site is published at https://[username].github.io/[repository-name]/` 메시지 확인

5. **완료! 🎉**
   - 제공된 URL로 접속하여 학습 플랫폼 사용
   - URL을 북마크하거나 친구들과 공유하세요!

### 방법 2: Git 사용 (개발자용)

```bash
# 저장소 클론 (또는 새로 만들기)
git clone https://github.com/[username]/electrician-study.git
cd electrician-study

# 파일 추가
cp /path/to/files/* .

# 커밋 및 푸시
git add .
git commit -m "전기기사 학습 플랫폼 초기 버전"
git push origin main

# GitHub Pages 설정은 웹 인터페이스에서 진행
```

## 🌟 주요 특징

### 데이터 구조
각 개념은 다음 정보를 포함합니다:
- **제목**: 개념 이름 (한글/영문)
- **정의**: 명확하고 간결한 설명 (2-3줄)
- **공식**: 관련 수식 (해당 시)
- **출제빈도**: 상/중/하 (실제 시험 출제 경향)
- **예시**: 실용적인 예제
- **관련 개념**: 연관된 다른 개념 링크

### 학습 기능
- ✅ **학습 완료 체크**: 공부한 개념 표시
- ⭐ **즐겨찾기**: 중요한 개념 북마크
- 📊 **진도율 추적**: 전체 학습 진행률 실시간 확인
- 🔄 **정렬 옵션**: 기본순/이름순/빈도순

### 사용자 경험
- 🌓 **다크모드**: 밤 시간 학습에 최적화
- 📱 **모바일 우선**: 이동 중에도 편한 학습
- ⚡ **빠른 속도**: 즉각적인 반응속도
- 💾 **자동 저장**: 학습 기록 자동 보존

## 📱 사용 방법

### 1. 개념 검색
- 상단 검색창에서 원하는 키워드 입력
- 제목, 정의, 공식에서 모두 검색됩니다

### 2. 필터 활용
- **카테고리**: 과목별로 보기
- **출제빈도**: 중요도가 높은 개념 우선 학습
- **보기**: 즐겨찾기나 학습 상태로 필터링

### 3. 학습 관리
- ⭐ 아이콘: 중요한 개념 즐겨찾기
- ✅ 아이콘: 학습 완료 표시
- 대시보드에서 학습 진도 확인

### 4. 상세 보기
- 개념 카드 클릭 → 상세 정보 모달
- 공식, 예시, 관련 개념 확인
- 관련 개념 태그 클릭 → 자동 검색

## 🎓 학습 팁

### 효과적인 학습 전략
1. **빈도 상인 개념부터**: 출제빈도가 높은 개념 우선 학습
2. **매일 반복 학습**: 하루 10-20개 개념씩 꾸준히
3. **관련 개념 연결**: 관련 개념 태그를 활용한 입체적 학습
4. **즐겨찾기 활용**: 어려운 개념은 즐겨찾기로 반복 복습
5. **진도율 확인**: 대시보드에서 학습 진행 상황 모니터링

### 시험 직전 활용법
- 즐겨찾기한 개념 집중 복습
- 미학습 개념 필터로 빠진 부분 체크
- 빈도 상 개념 최종 점검
- 모바일로 이동 중에도 학습
